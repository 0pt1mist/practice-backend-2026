# Booking API (Coworking Space)

REST API для системы бронирования ресурсов в коворкинге (рабочие места, переговорные комнаты) с проверкой конфликтов времени, системой отзывов и JWT-аутентификацией.

## Предметная область

Система позволяет пользователям бронировать рабочие места (desks) и переговорные (rooms). Главная особенность — строгая проверка временных интервалов: система не позволяет забронировать один и тот же ресурс на пересекающееся время.

Основные правила:
- Пользователь может бронировать ресурсы и просматривать свои бронирования.
- Администратор управляет списком ресурсов.
- Отзывы можно оставлять только после завершения времени бронирования.
- Поиск позволяет найти только те ресурсы, которые свободны в указанный интервал.

## Стек

- **Runtime:** Node.js 22
- **Framework:** Express.js
- **ORM:** Prisma
- **Database:** PostgreSQL
- **Auth:** JWT (JSON Web Token)
- **Testing:** Jest + Supertest

## Реализованные возможности

- Регистрация и логин пользователей (роли ADMIN и USER).
- JWT Bearer аутентификация и защита роутов.
- CRUD для ресурсов (только для администратора).
- Создание бронирований с автоматической проверкой на "нахлесты" времени.
- Отмена бронирований и просмотр истории (профиль пользователя).
- Поиск свободных ресурсов на заданную дату и время.
- Фильтрация ресурсов по типу (стол/комната) и минимальной вместимости.
- Система отзывов с расчетом среднего рейтинга ресурса в режиме реального времени.
- Автоматические тесты ключевых сценариев.

## Запуск

1. Установить зависимости:

2. Применить миграции:

## Основные эндпоинты

### Аутентификация
- `POST /api/auth/register` — Регистрация
- `POST /api/auth/login` — Вход

### Ресурсы
- `GET /api/resources` — Список ресурсов с фильтрами (`?type=room&minCapacity=5`)
- `POST /api/resources` — Создание (Admin only)
- `GET /api/resources/search` — Поиск свободных на интервал (`?start=...&end=...`)

### Бронирование
- `POST /api/bookings` — Создание брони
- `GET /api/bookings/my` — История бронирований текущего пользователя
- `PATCH /api/bookings/:id/cancel` — Отмена бронирования

### Отзывы
- `POST /api/resources/:id/reviews` — Оставить отзыв

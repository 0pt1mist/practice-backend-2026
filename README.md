# Survey API — Сервис опросов и голосований

REST API для создания, управления и прохождения опросов. Система поддерживает разграничение прав доступа, управление жизненным циклом опроса (черновик -> опубликован -> закрыт) и базовую аналитику результатов.

## Стек технологий
- **Язык:** TypeScript
- **Платформа:** Node.js
- **Фреймворк:** Express.js
- **ORM:** Prisma
- **База данных:** PostgreSQL
- **Аутентификация:** JWT (JSON Web Token)

## Роли и возможности

### RESPONDENT (Респондент)
- Просмотр списка опубликованных опросов.
- Прохождение опросов (отправка ответов).
- Защита от повторного прохождения (один пользователь — один ответ на опрос).

### AUTHOR (Автор)
- Создание и редактирование опросов (в статусе «Черновик»).
- Управление статусами: Публикация (Черновик → Опубликовано) и Закрытие (Опубликовано → Закрыт).
- Просмотр аналитики: количество ответов, процентное распределение.
- Экспорт результатов в формате JSON.
- Удаление и архивация собственных опросов.

 API Эндпоинты
 Авторизация и Профиль
Метод	Путь	Описание	Доступ
POST	/auth/register	Регистрация пользователя	Публичный
POST	/auth/login	Вход в систему (получение JWT)	Публичный
POST	/auth/logout	Выход из системы	Авторизован
GET	/profile	Информация о текущем пользователе	Авторизован
PATCH	/profile	Обновление данных профиля	Авторизован
 Опросы (Респондент)
Метод	Путь	Описание	Доступ
GET	/surveys	Список опубликованных опросов	Публичный
GET	/surveys/:id	Детали конкретного опроса	Публичный
GET	/surveys/:name	Поиск по уникальному имени	Публичный
POST	/surveys/:id/submit	Отправка ответов на опрос	RESPONDENT
GET	/surveys/:id/status	Проверка (пройден ли опрос юзером)	RESPONDENT
 Управление опросами (Автор)
Метод	Путь	Описание	Доступ
GET	/surveys/my	Список опросов автора	AUTHOR
POST	/surveys	Создание опроса (Черновик)	AUTHOR
PATCH	/surveys/:id	Редактирование вопросов и текста	AUTHOR (Owner)
POST	/surveys/:id/publish	Перевод в "Опубликовано"	AUTHOR (Owner)
POST	/surveys/:id/close	Перевод в "Закрыт"	AUTHOR (Owner)
DELETE	/surveys/:id	Удаление/архивация опроса	AUTHOR (Owner)
 Аналитика и Результаты (Автор)
Метод	Путь	Описание	Доступ
GET	/surveys/:id/analytics	Статистика по ответам в %	AUTHOR (Owner)
GET	/surveys/:id/export	Экспорт результатов в JSON	AUTHOR (Owner)
GET	/surveys/:id/responses	Список детальных ответов	AUTHOR (Owner)
